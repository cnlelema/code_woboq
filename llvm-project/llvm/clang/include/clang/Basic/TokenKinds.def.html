10// tok::ampamp (corresponding to the && token) as well as keywords for various
11// languages. Users of this file must optionally #define the TOK, KEYWORD,
12// CXX11_KEYWORD, CONCEPTS_KEYWORD, ALIAS, or PPKEYWORD macros to make use of
13// this file.

233// C99 6.4.1: Keywords. These turn into kw_* tokens.
234// Flags allowed:
235// KEYALL - This is a keyword in all variants of C and C++, or it
236// is a keyword in the implementation namespace that should
237// always be treated as a keyword
238// KEYC99 - This is a keyword introduced to C in C99
239// KEYC11 - This is a keyword introduced to C in C11
240// KEYCXX - This is a C++ keyword, or a C++-specific keyword in the
241// implementation namespace
242// KEYNOCXX - This is a keyword in every non-C++ dialect.
243// KEYCXX11 - This is a C++ keyword introduced to C++ in C++11
244// KEYCXX2A - This is a C++ keyword introduced to C++ in C++2a
245// KEYCONCEPTS - This is a keyword if the C++ extensions for concepts
246// are enabled.
247// KEYMODULES - This is a keyword if the C++ extensions for modules
248// are enabled.
249// KEYGNU - This is a keyword if GNU extensions are enabled
250// KEYMS - This is a keyword if Microsoft extensions are enabled
251// KEYMSCOMPAT - This is a keyword if Microsoft compatibility mode is enabled
252// KEYNOMS18 - This is a keyword that must never be enabled under
253// MSVC <= v18.
254// KEYOPENCLC - This is a keyword in OpenCL C
255// KEYOPENCLCXX - This is a keyword in OpenCL C++
256// KEYNOOPENCL - This is a keyword that is not supported in OpenCL C
257// nor in OpenCL C++.
258// KEYALTIVEC - This is a keyword in AltiVec
259// KEYZVECTOR - This is a keyword for the System z vector extensions,
260// which are heavily based on AltiVec
261// KEYBORLAND - This is a keyword if Borland extensions are enabled
262// KEYCOROUTINES - This is a keyword if support for C++ coroutines is enabled
263// BOOLSUPPORT - This is a keyword if 'bool' is a built-in type
264// HALFSUPPORT - This is a keyword if 'half' is a built-in type
265// WCHARSUPPORT - This is a keyword if 'wchar_t' is a built-in type
266// CHAR8SUPPORT - This is a keyword if 'char8_t' is a built-in type
267//
